Instalar Raspbian. Configurando Raspberry PI 1

Oct 18, 2013
raspberry

Desde la web de Raspberry descargamos el fichero de Raspbian y el instalador Win32DiskImager.
Lo instalamos en Windows y con la SD arrancamos la Raspberry.

Miramos su IP mediante el router y conectamos:
ssh pi@IP_raspberry
Y ejecutamos el comando:
sudo raspi-config

Ahí redimensionamos el sistema y creamos un usuario para nosotros.
