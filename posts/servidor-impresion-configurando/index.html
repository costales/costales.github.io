Servidor impresión. Configurando Raspberry PI 9

Oct 22, 2013
raspberry

Instalamos servidor:
sudo apt-get install cups

Damos permisos para usar cups:
sudo usermod -a -G lpadmin tu_usuario

Abrimos firewall:
sudo ufw allow 631

Añadimos nuestra LAN:
`sudo vi /etc/cups/cupsd.conf

#Listen localhost:631
Listen *:631
Listen /var/run/cups/cups.sock

Restrict access to the server…


  Order allow,deny
  Allow 192.168.1.*

Restrict access to the admin pages…


  Order allow,deny
  Allow 192.168.1.*

Restrict access to configuration files…

<location /admin/conf>
  AuthType Default
  Require user @SYSTEM
  Allow 192.168.1.*
  Order allow,deny
`

Reinciamos servidor:
sudo service cups restart

Accedemos a la URL: https://nuestra_raspberry_IP:631 y añadimos la impresora marcando “Share this printer”.
