Servidor web. Configurando Raspberry PI 4

Oct 18, 2013
raspberry server

Instalamos todo a la vez (servidor apache, base de datos mysql y servidor de email):

`sudo apt-get install apache2 php5 libapache2-mod-auth-mysql php5-mysql mysql-server phpmyadmin` `postfix`
```Nos pedirá las contraseñas para root de mysql y phpmyadmin. Y el dominio del servidor de correo.  
  
Ya deberiamos poder acceder a:  
https://IP\_Raspberry  
https://IP\_Raspberry/phpmyadmin  
  
Activar redirecciones mediante .htaccess:

sudo a2enmod rewrite ; sudo service apache2 restart
