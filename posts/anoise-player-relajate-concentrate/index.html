ANoise Player: Relájate | Concéntrate

Apr 3, 2015
apps

Mi nueva aplicación es ANoise, que permite integrar sonidos ambientales en el área de notificación de Ubuntu:
Cafetería | Fuego | Bosque | Noche | Lluvia | Mar | Tormenta | Viento

ANoise Player

Cómo instalar: Ejecuta estos comandos en una Terminal:

sudo add-apt-repository ppa:costales/anoise  
sudo apt-get update  
sudo apt-get install anoise  

```Y busca _Sonido ambiental_ en tu lanzador.  
Las próximas veces, simplemente accede al área de notificación para controlarlo.  
  
También puedes instalar ambientes de otros usuarios con el comando:

sudo apt-get install anoise-community-extension1

  
[+info](https://anoise.tuxfamily.org/)
