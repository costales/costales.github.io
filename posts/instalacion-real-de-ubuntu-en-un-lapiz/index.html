Instalación real de Ubuntu en un lápiz USB

Jan 13, 2012
software usb
  • Quemamos un USB o CD Live.
  • Desconectamos el disco duro para evitar que la instalación sobreescriba el GRUB actual.
  • Lo arrancamos con el USB final que queremos que tenga la instalación de Ubuntu.
  • Instalamos en el USB como disco duro.
  • Ejecutamos este comando para crear el GRUB en el USB:
  • sudo grub-install –root-directory=/media/path_usb /dev/sdb
  • Hay que cambiar los path_usb y sdb por el que corresponda.
  • Reiniciamos y todo debería de funcionar correctamente.

Fuente: Ask Ubuntu.
