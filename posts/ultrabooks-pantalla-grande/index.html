Ultrabooks con pantalla grande

Dec 1, 2019
hardware

Hay muchos ultrabooks de 13,3” o menos, pero hay muy pocos que podamos comprar con 14 pulgadas o más.

Ya que tuve que realizar la elección para este Black Friday, os voy a poner las alternativas, los puntos importantes y por cual me decanto.

Los requisitos son:

  • Peso <= 1Kg
  • CPU >= i5
  • RAM >= 8GB
  • Pantalla >= 14”

Las opciones actuales en el mercado son:

  • LG Gram
  • Acer swift 5
  • Slimbook X PRO
  • Lenovo Carbon X1

Pulgadas

LG es el único que permite comprar más de 14” manteniendo el peso.

Precio

El Acer es el único que baja claramente de los 1000€, aunque sólo se puede comprar con un disco duro de 256GB.

Linux

Slimbook será el más compatible con Ubuntu.

Gama

Sólo el modelo de Lenovo es gama empresarial, muy lejos en cuanto a calidad de la gama doméstica del resto.

Decisión personal

LG lo descarto por el tamaño de las teclas, pues el teclado es una de las partes más importantes (para mi).

Si el precio no importa, sin duda el Lenovo por el mero hecho de ser gama profesional.

Acer si el precio es determinante.

Slimbook si vamos a usar una distribución Linux, por su compatibilidad.

Si necesitas personalizar el hardware del portátil, Lenovo y Slimbook son las mejores opciones.
