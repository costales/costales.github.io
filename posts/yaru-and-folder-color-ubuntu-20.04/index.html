Yaru & Folder Color in Ubuntu 20.04

Apr 22, 2020
apps planet customization

The new Yaru in Ubuntu 20.04 is awesome! I love it!

But there isn't an icon theme working fine with Folder Color & Yaru… until now. Just in time!

Thanks to Jan Schröder with the Yaru-Colors project you can enjoy it now!

How to install it

In Ubuntu 20.04 run these commands from a Terminal (from code is available here):

sudo apt install folder-color
sudo add-apt-repository ppa:costales/yaru-colors-folder-color
sudo apt install yaru-colors-folder-color
nautilus -q
