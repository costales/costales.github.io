Transmission desde el móvil - Configurando Raspberry 5

Oct 18, 2013
raspberry transmission

Instalamos:

sudo apt-get install transmission-daemon

Y configuramos:

mkdir /tu\_path/progress  
mkdir /tu\_path/completed  
sudo usermod -a -G debian-transmission tu\_usuario  
chgrp debian-transmission /tu\_path/progress  
chgrp debian-transmission /tu\_path/completed  
chmod 770 /tu\_path/progress  
chmod 770 /tu\_path/completed  
sudo service transmission-daemon stop  
sudo adduser tu\_usuario debian-transmission  
sudo nano /etc/init.d/transmission-daemon  
sudo chown tu\_usuario -R /var/lib/transmission-daemon/info/  
sudo chown tu\_usuario -R /tu\_path/progress  
sudo chown tu\_usuario -R /tu\_path/completed  
sudo chown tu\_usuario -R /etc/transmission-daemon/settings.json

Cambiamos en el fichero estos parámetros:

sudo nano /etc/transmission-daemon/settings.json  
  
"rpc-whitelist": "127.0.0.1" > "rpc-whitelist": "\*.\*.\*.\*",  
"rpc-password": "password" > "rpc-password": "tu\_clave",  
"rpc-username": "username" > "rpc-username": "tu\_usuario",  
"download-dir": "/var/lib/transmission-daemon/downloads" > "download-dir": "/tu\_path/completed"  
"incomplete-dir": "/var/lib/transmission-daemon/downloads" > "incomplete-dir": "/tu\_path/progress"

Y reiniciamos el servicio:

service transmission-daemon start

Visitamos:

https://raspberry_IP:9091

Metemos el usuario y clave del fichero de configuración.

En el móvil instalamos Transmission Remote y lo configuramos con los parámetros de nuestra red ;) Recordar abrir los puertos en el router para acceder desde el exterior.
