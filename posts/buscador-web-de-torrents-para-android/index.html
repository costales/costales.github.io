Buscador web de Torrents para Android

Jul 29, 2013
apps transmission

Uso el excelente Remote Transmission para desde el móvil controlar las descargas, pero con un buscador de torrents, el conjunto es perfecto, ya que desde el móvil puedes buscar, descargar el torrent que automáticamente se carga en el Remote Transmission.

¿Cual es el problema? Que todos los buscadores de torrent en la Play Store son bastante malos o directamente no funcionan.
Intenté encontrar un buscador web que tenga vista móvil, pero no encontré ninguno.

Por suerte, existe este proyecto: Mobile-Web-Torrent. Copìamos el fichero .php a nuestro servidor y si queremos protegemos el acceso por clave a esa web.

Sencillo, bien adaptado y funciona perfectamente :)
