Mozilla Firefox en Ubuntu en tu idioma

Aug 2, 2014
software

Por ejemplo, en Ubuntu 14.04, Firefox no está en asturiano, quejándose de que esa lengua no está disponible para esa versión. Pero yo sé que está ;)

Instalando el paquete de idioma desde el repositorio oficial de Mozilla aparecerá en lengua asturiana. ¿Cómo lo hacemos? Ejecutando este comando en la Terminal:

sudo add-apt-repository ppa:ubuntu-mozilla-security/ppa  
sudo apt-get update && sudo apt-get install firefox-locale-ast -y

Hay que cambiar “-ast” por tu código de lengua ISO ;)
