How fix 100% CPU usage in Ubuntu 11.10? Jan 14, 2012 In my desktop and laptop (HP nx7300) with Intel, I have big problems with Ubuntu 11.10. The system occasionally hangs with 100% cpu usage. For fix this I ran in a Terminal: sudo apt-get install compizconfig-settings-manager ccsm Search openGL, and disable all checkboxs. Reboot. Enjoy again this version :) Found in Launchpad, but I don't remember the bug number.
