How fix 100% CPU usage in Ubuntu 11.10? Jan 14, 2012 In my desktop and laptop (HP nx7300) with Intel, I have big problems with Ubuntu 11.10. The system occasionally hangs with 100% cpu usage. For fix this I ran in a Terminal: sudo apt-get install compizconfig-settings-manager ccsm Search openGL, and disable all checkboxs. Reboot. Enjoy again this version :) Found in Launchpad, but I don't remember the bug number.
Instalación real de Ubuntu en un lápiz USB Jan 13, 2012 Quemamos un USB o CD Live. Desconectamos el disco duro para evitar que la instalación sobreescriba el GRUB actual. Lo arrancamos con el USB final que queremos que tenga la instalación de Ubuntu. Instalamos en el USB como disco duro. Ejecutamos este comando para crear el GRUB en el USB: sudo grub-install –root-directory=/media/path_usb /dev/sdb Hay que cambiar los path_usb y sdb por el que corresponda.
