Convertir vídeo de Youtube a .avi con buena calidad Feb 28, 2012 Desde la Terminal de Ubuntu necesitamos instalar 2 programas: sudo apt-get install youtube-dl mencoder Ahora para descargar un video de Youtube: youtube-dl dirección_web_del_video_incluyendo_http Para convertir el vídeo descargado a .avi con buena calidad: mencoder video.flv -ovc lavc -oac mp3lame -o video.avi Vía Ubuntu Forums.
