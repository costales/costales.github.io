Mozilla Firefox en Ubuntu en tu idioma Aug 2, 2014 Por ejemplo, en Ubuntu 14.04, Firefox no está en asturiano, quejándose de que esa lengua no está disponible para esa versión. Pero yo sé que está ;) Instalando el paquete de idioma desde el repositorio oficial de Mozilla aparecerá en lengua asturiana. ¿Cómo lo hacemos? Ejecutando este comando en la Terminal: sudo add-apt-repository ppa:ubuntu-mozilla-security/ppa sudo apt-get update && sudo apt-get install firefox-locale-ast -y Hay que cambiar “-ast” por tu código de lengua ISO ;)

Raspberry PI: Backup SD card with dd command under Linux Jun 7, 2014 First, know your partitions: fdisk -l Full backup of the image: dd if=/dev/sdx of=/path/to/image ```or for compression (recommended): dd if=/dev/sdx | gzip > /path/to/image.gz To restore the backup you reverse the commands:``` dd if=/path/to/image of=/dev/sdx ```or when compressed: gzip -dc /path/to/image.gz | dd of=/dev/sdx The info is from [here](https://raspberrypi.stackexchange.com/a/312).
