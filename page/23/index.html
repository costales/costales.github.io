Añadir disco duro externo. Configurando Raspberry PI 3 Oct 18, 2013 Vamos añadir un disco duro externo para las descargas que hagamos. Lo formateamos en ext3. Creamos su punto de montaje y poner permiso para todos los usuarios: sudo mkdir -p /tu\_path/HD ; sudo chown tu\_usuario:tu\_usuario -R /tu\_path/HD ; sudo chmod 700 -R /tu\_path/HD y lo añadimos en el arranque, averiguando primero su SUIID: sudo blkid sudo vi /etc/fstab añadiendo esta línea: UUID=6a60524d-061d-454a-bfd1-38989910eccd /tu_path/HD ext3 defaults 0 0 Reiniciamos y comprobamos que está montado: ls /tu_path/HD
Tu propio usuario. Configurando Raspberry PI 2 Oct 18, 2013 Añade tu usuario: sudo adduser tu_usuario y edita sudo vi /etc/group Cambiando los “pi” por tu usuario. Cierra la sesión ssh y vuelve a acceder con tu usuario. Y borra “pi”: sudo userdel -r pi Aprovechamos a actualizar el sistema y probar sudo: sudo apt-get update ; sudo apt-get upgrade ; sudo apt-get clean
